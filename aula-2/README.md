dir - Exibe os diretórios
cd - Abre um diretório ou volta ao diretório anterior
md - Cria um diretório
rd - Remove um diretório
cls - Limpa a janela
echo %errorlevel% - Exibe o retorno do último aplicativo executado
ipconfig - Exibe as configurações de endereço de IP
tasklist - Exibe a lista de processos sendo executados no windows
taskkill - Finaliza um processo que está sendo executado
copy - Copia um arquivo para outro local
del - Exclui um arquivo
move - Move um arquivo para outro local
help - Exibe informações de ajuda sobre comandos
tree - Exibe graficamente a estrutura de diretórios de uma unidade ou caminho
title - Define um título para a janela do cmd em execução
systeminfo - Exibe as propriedades e configurações da máquina
rename - Renomea um arquivo específico
ren - Renomea um arquivo específico
findstr - Procura cadeira de caractéres em um arquivo
exit - Sai do cmd
color - Modifica a cor da fonte do cmd
