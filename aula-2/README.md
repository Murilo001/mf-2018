dir - Exibe os diretórios<br />
cd - Abre um diretório ou volta ao diretório anterior<br />
md - Cria um diretório<br />
rd - Remove um diretório<br />
cls - Limpa a janela<br />
echo %errorlevel% - Exibe o retorno do último aplicativo executado<br />
ipconfig - Exibe as configurações de endereço de IP<br />
tasklist - Exibe a lista de processos sendo executados no windows<br />
taskkill - Finaliza um processo que está sendo executado<br />
copy - Copia um arquivo para outro local<br />
del - Exclui um arquivo<br />
move - Move um arquivo para outro local<br />
help - Exibe informações de ajuda sobre comandos<br />
tree - Exibe graficamente a estrutura de diretórios de uma unidade ou caminho<br />
title - Define um título para a janela do cmd em execução<br />
systeminfo - Exibe as propriedades e configurações da máquina<br />
rename - Renomea um arquivo específico<br />
ren - Renomea um arquivo específico<br />
findstr - Procura cadeira de caractéres em um arquivo<br />
exit - Sai do cmd<br />
color - Modifica a cor da fonte do cmd
